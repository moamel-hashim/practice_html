TCP
Transmission Control Protocol is a method (protocol) used along with the Internet Protocol (IP) to send data in the form of message units, called packets, between computers over the Internet.
IP
Internet Protocol is the method or protocol by which data ios sent from one computer to another one the Internet. Each computer to another on the Internet. Each computer on the Internet is uniquely identified by an IP address.
FTP
File Transfer Protocol is a protocol used to exchange files between computers on the Internet.
HTTP
Hyper Transfer Protocol is the protocol used for exchanging text, graphic images, sounds, video, and other multimedia files on the Web.
